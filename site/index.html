How We Scored

Data Sources

  • MOM Occupational Wage Survey 2024 — ~560 occupations with SSOC codes, gross monthly wage percentiles
  • SkillsFuture Skills Framework — Task-level data per role across 38 sectors, including key tasks and technical skills

Scoring Model

Each occupation was scored by Claude Haiku 4.5 (temperature 0.2) via the Anthropic Batch API. Two independent runs were averaged for stability.

Scale Anchors (0-10)

  • 0-1: Physical, manual labor — cleaner, refuse collector
  • 2-3: Interpersonal + physical — hairdresser, bus driver
  • 4-5: Mixed — nurse, teacher, property agent
  • 6-7: Knowledge work — accountant, HR manager
  • 8-9: Digital/cognitive — software engineer, translator
  • 10: Routine digital — data entry clerk

Key Differentiator

Unlike title-only scoring, we provide the LLM with task-level SkillsFuture data for each occupation, enabling more granular and accurate assessment of which specific tasks are automatable.
